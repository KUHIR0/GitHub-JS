GitHubga Yuklash

kodni yozib tayyor qilgach Terminalni ishga tushirib quyidagi buyruqlar orqali githubga joylaymiz:

    #1. git init  
    #2. git status  
    #3. git add -A  
    #4. git commit -m "......."  
    #5. GitHubga  
      git remote add origin    https://github.com/KUHIR0/GitHub-JS.git
      git branch -M main
      git push -u origin main

Commit qo`shish

Yozgan kodimizga nimadur o`zgarish qilsak usha o`zgarishni GitHubga yuklash u.n quyidagi buyruqlarni bajarimiz:

      #1. git add -A  
      #2. git commit -m "......."  
      #3. git push 
      #4. git log

GitHubdan boshqalarning kodini kompiyuterimizga obkelish 

      #1. Visul Studio code muharirida papka ochamiz
      #2. Terminalda cd ....(nuqtalar urniga yaratilgan papka nomini kiritamiz)
      #3. git clone ..... (nuqtalar urniga githubdagi faylimiz joylashgan manzil, nomi kiritiladi. M: https://github.com/KUHIR0/GitHub-JS.git)

           qo`shimcha buyruqlar
      #4. cd ( cd va papka nomini yozib usha papkani ichiga kirish)
      #5. cd ../ (joriy papkadn 1ta orqaga chiqish)

Ofisda yozgan kodimizni uydagi kompiyuterda ham ochish u.n

#1. cd ....(papka nomi) #2. git pull (githubdagi faylimizni kompiyuterimizga yuklab beradi)
